# Roadmap

> The plan for making AI Security Hub the definitive AI security resource.

---

## Phase 1 — Foundation (Current)

**Goal: Launch with real depth, not just links**

- [x] Complete README with all major sections
- [x] Payload collection (SecLists-style)
- [x] Full cheat sheet set (5 sheets)
- [x] Labs catalog with DVAP featured
- [x] Tools catalog (100+ entries planned)
- [x] Research database
- [x] CTF catalog
- [x] Learning paths (beginner → expert)
- [x] MCP Security dedicated section
- [x] Community infrastructure (CONTRIBUTING, CoC, templates)
- [x] GitHub Actions link checker

**Target: 100 stars**

---

## Phase 2 — Community Growth

**Goal: Become the go-to reference for AI security professionals**

- [ ] 50+ payload entries across all categories
- [ ] Video walkthrough of DVAP integration
- [ ] Monthly "AI Security Digest" in GitHub Discussions
- [ ] Dataset section (training data attack datasets)
- [ ] Benchmarks section (evaluation framework catalog)
- [ ] Multilingual content (Chinese, Spanish, Portuguese translations)
- [ ] Twitter/X presence with payload-of-the-week

**Target: 500 stars**

---

## Phase 3 — Authority

**Goal: Become the canonical reference, cited by researchers and teams**

- [ ] Research paper submission tracker (track new AI security papers)
- [ ] Annual "State of AI Security" report
- [ ] Contributor leaderboard
- [ ] AI security news aggregation
- [ ] Partnerships with AI security labs and conferences
- [ ] Inclusion in security course syllabi

**Target: 1,000+ stars**
