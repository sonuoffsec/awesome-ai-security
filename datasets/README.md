# AI Security Datasets

> Curated datasets for AI security research, training, and evaluation.

---

## Prompt Injection Datasets

| Dataset | Description | Size | Link |
|---|---|---|---|
| **HackAPrompt Dataset** | 600k+ prompt injection attempts from the HackAPrompt competition | 600k examples | [HuggingFace](https://huggingface.co/datasets/hackaprompt/hackaprompt-dataset) |
| **Jailbreak Prompts** | Collected jailbreak prompts from public sources | Varies | [GitHub](https://github.com/verazuo/jailbreak_llms) |
| **JailbreakBench Dataset** | Standardized jailbreak evaluation dataset | 100 behaviors | [GitHub](https://github.com/JailbreakBench/jailbreakbench) |
